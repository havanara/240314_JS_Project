함수(function) : 특정 기능을 수행하기 위해 작성한 코드 - 호출되어야만 실행 - 자바에서의 메서드처럼 호출하여 실행 - 자바에서는 반드시 class가 만들어져야하고 그 안에 method 생성 - but! 자바스크림트는 어디서든 function을 생성할 수 있음

  • function 이라는 키워드를 사용하여 정의
  • function 함수명(파라미터){ 로직작성; return 값; }
  • 함수명 : 기능을 함축하는 의미로 작성
  • 파라미터(매개변수) : 선택적으로 작성(없어도 () 괄호는 반드시 있어야 함)
  • 자바스크립트에는 타입이 없어서 파라미터의 타입도 없이 때문에 자유도가 높아서 다양한 형태로 사용 가능 다양한 형태로 매개변수를 받을 수 있음
  • return 함수종료 역할과 동시에 특정 결과값을 호출한 위치로 반환
