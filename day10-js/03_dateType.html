변수나 상수에 할당할 수 있는 값의 종류

  • 숫자 : Number -> 정수, 실수, NaN(Not a Number의 약자 : 숫자가 아니라는 뜻)
  • 문자 : String -> "", '', `` 으로 감싼 값들을 문자로 인지
  • 불리언 : Boolean -> true, false
  • 객체 : Object -> 1개 이상의 데이터가 특정 구조를 가지고 있는 형태 {키 : 값} {} : 비어있는 아무것도 없는 사실도 1개의 객체로 취급
  • 배열 : Array -> [1,2,3,4] 형태의 집합 (java의 List와 비슷, 순서와 길이(length)가 있음)
  • 알수없는 값 : null -> 값이 없다는게 아니라 인지가 안된다는 의미
  • 미지정된 값 : undefinde -> 값이 할당되지 않음(값이 안왔어~!)
  1. 기본자료형 : 값이 메모리의 주소값을 가르키는 곳에 직접 할당이 되어있는 값 number, boolean, string(참조자료형이지만 편의상 기본자료형과 같이 사용)
  2. 참조자료형 : 값이 저장되어 있는 주소를 할당 string, array, object, function, null, undifind
  3. == 같다(안에 들어가 있는 값만 비교) != 같지 않다 === 같다(값과 타입이 같은지 비교) !=== 같지않다(값과 타입이 같지 않은지 비교)
