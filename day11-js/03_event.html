이벤트 객체는 전역객체로서 모든 태그에 참조시킬 수 있다.

  • a tag / button tag에 이벤트를 주로 참조시켜 사용 클릭에 대한 반응을 기대할 수 있게 만드는게 일반적
  • a 태그는 href 속성에 의해 자체이동(동기화 방식)이 구현되어 있어서 별도의 이동을 막는 코드가 필요
  • 객체에 이벤트를 부여한느 방식 크게 3가지
  • 각자 목적과 작동방식이 다름
  • 화면의 출력 영역을 document영역(객체)라고 한다
  • document영역을 DOM 영역이라고 칭함
  • tag는 html의 element(엘리먼트), 속성은 attribute, 일반 텍스트 textNode라고 함
  • html로 출력하기 : innerHTML이라는 property를 사용
  • text로 출력하기 : innerText property를 사용
  • html element를 가져올 때 id로 가져오는 방법(일반적) getElementById('idName');
  • html element를 가져올 때 class로 가져오는 방법 querySelector('selector') -> # . 선택자 가능
